## 一、核心术语说明

| 术语 | 核心释义 |
|------|----------|
| **Token** | 大模型处理文本的最小单位。约 1000 个汉字 ≈ 1300 个 Token，是 API 计费的核心计量单位。 |
| **上下文窗口** | 模型单次可处理的最大文本长度，决定能否一次性读取长文档、完整代码库等。 |
| **MoE 架构** | 混合专家模型（Mixture of Experts），仅激活部分参数完成推理，兼顾大参数量能力与低推理成本。 |
| **开源协议** | 决定模型能否商用、二次开发的核心规则。主流宽松协议为 Apache 2.0、MIT，商用限制少。 |
